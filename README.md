# 🚀 Деплой на GitHub Pages

## Быстрый старт

1. Создайте репозиторий на GitHub: `mmmatveyz.github.io` (или любой другой)
2. Загрузите файлы проекта:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/mmmatveyz/mmmatveyz.github.io.git
   git push -u origin main
